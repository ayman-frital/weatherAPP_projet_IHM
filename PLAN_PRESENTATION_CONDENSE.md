## 06 - INTÉGRATION API

**Titre**
🌐 Intégration OpenWeatherMap API

**À Propos API**
- ✅ Leader météo mondiale
- ✅ Données temps réel
- ✅ Couverture globale
- ✅ Fiabilité 99.9%

**Endpoints Utilisés**
```
1. Weather Current
   GET /weather?q={city}
   
2. Forecast 5 Days
   GET /forecast?q={city}
   
3. Geocoding
   GET /geo/1.0/direct?q={city}
```

**Flux Données**
```
Utilisateur
  ↓ (Recherche)
Application
  ↓ (Validation)
Service API
  ↓ (HTTP Request)
OpenWeatherMap
  ↓ (Réponse JSON)
Parsing Gson
  ↓ (Désérialisation)
Model Objects
  ↓ (Mise à jour)
Affichage UI
  ↓
✅ Données visibles
```

**Gestion Erreurs**
- Try-catch robuste
- Custom exceptions
- Reconnexion automatique
- Messages clairs
