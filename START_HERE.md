**Don't have them?** Follow SETUP_INSTRUCTIONS.md

2. Edit: `src/main/java/com/weather/service/OpenWeatherService.java`
3. Replace: `private static final String API_KEY = "YOUR_API_KEY_HERE";`
4. With your actual API key

### Step 3: Build & Run
```bash
mvn clean package
mvn javafx:run
```

## 📞 Quick Help

### Problem: Application won't start
**Solution**: Rebuild project
```bash
mvn clean package
mvn javafx:run
```

### Problem: "Invalid API key" error
**Solution**: 
1. Check your API key in OpenWeatherService.java
2. Verify it's active on OpenWeatherMap.org
3. Test: `https://api.openweathermap.org/data/2.5/weather?q=London&appid=YOUR_KEY`

### Problem: "City not found"
**Solution**: Use official city names (e.g., "London", "Paris", "New York")

### Problem: Maven command not found
**Solution**: 
1. Check Maven installation: `mvn -version`
2. Restart terminal
3. Add Maven to PATH if needed
